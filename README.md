# Weather App

## Description
A web app to fetch current weather information based on user input using a public API.

## Features
- Search weather by city
- Display temperature, humidity, and weather description
- Responsive design

## Technologies Used
- HTML
- CSS
- JavaScript
- Weather API (like OpenWeatherMap)

## Installation
# Clone the repository
git clone https://github.com/yourusername/weather-app.git

# Open the project
cd weather-app
# Open index.html in your browser
# Add your API key in script.js if required
